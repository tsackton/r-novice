### Challenge {.challenge}

* What are the values after each statement in the following?
 

```r
# Answers are in the commented part of each line
mass <- 47.5           # mass = 47.5
age  <- 122            # age = 122
mass <- mass * 2.0     # mass = 95
age  <- age - 20       # age 102
massIndex <- mass/age  # massIndex = 0.9313725
```
